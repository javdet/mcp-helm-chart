MCP Helm Chart

Universal Helm chart for deploying MCP servers in direct or proxy (supergateway) mode

⚠ Distribution changed: Starting from v0.4.1 the chart is published as an OCI artifact on GitHub Container Registry. The legacy helm repo add method is no longer updated.

Install OCI

# Helm 3.8+ required (OCI support)
helm install my-mcp oci://ghcr.io/javdet/charts/mcp \
  --version 0.4.1 \
  --namespace mcp \
  --create-namespace \
  -f values.yaml

Other commands

# Pull chart tarball
helm pull oci://ghcr.io/javdet/charts/mcp --version 0.4.1

# Show chart info
helm show all oci://ghcr.io/javdet/charts/mcp

# Template locally
helm template my-mcp oci://ghcr.io/javdet/charts/mcp --version 0.4.1 -f values.yaml

Helmfile

# helmfile.yaml
repositories:
  - name: mcp-chart
    url: ghcr.io/javdet/charts
    oci: true

releases:
  - name: my-mcp-server
    namespace: mcp
    chart: mcp-chart/mcp
    version: 0.4.1
    values:
      - values.yaml
